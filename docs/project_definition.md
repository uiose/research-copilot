# 项目定义
> 持续更新
## 1、项目目标
长期目标：
做一个面向初级研究者的 Research Copilot，能够根据用户提出的研究问题检索相关文献，并生成结构化的文献调研结果。

---

**当前目标：**
week01：
   完成一个最小原型baseline，跑通结构化问答，接入大语言模型，为LangGraph框架与检索工具做准备。


## 2、用户是谁
用户是初级研究者、研究生，或者刚进入某个方向、需要快速了解领域基本情况的人。


## 3、输入是什么
输入是一个研究主题或研究问题，通常表现为用户对某个领域、方法或方向的提问。

## 4、输出是什么
当前 baseline 版本的输出为结构化 JSON，包括：
- `topic`：当前处理的研究主题
- `assumptions`：当前版本的假设、边界和限制
- `next_step`：下一步建议动作或后续迭代方向


## 5、成功标准
- 程序可以运行
- 能接收一个研究主题作为输入
- 输出是固定格式的结构化 JSON
- 已建立固定评测题集
- 程序可以读取并检查题集字段

## 6、当前不做什么
- 暂时不接入工具检索
- 暂时不生成完整文献综述
- 暂时不做长期记忆
- 暂时不接入向量数据库
