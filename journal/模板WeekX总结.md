# Week X 总结

## 0) 模板说明
- 用途：本模板只做周级汇总与评价，引用周规划中的 `G1-G3` 和日记录中的证据/失败记录，不重新定义下周目标。
- 单向链路：周规划定义 -> 日记录引用并产证据 -> 周总结汇总并评价。
- YAML front matter：填写 `week` 等索引字段；共享词表也固定在 YAML 中，供后续程序读取。
- `[必填]` 字段构成最小闭环，`[可选]` 字段可以跳过。
- `目标对照` 只引用 `G#`，不重写周规划中的目标描述。
- `承接点（下周输入接口）` 是唯一允许前瞻到下周的区块，不写新周目标细节。
- `自由补充 [可选]` 仅保留上下文，不作为跨模板聚合字段。
